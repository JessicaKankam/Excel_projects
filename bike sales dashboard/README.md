# Bike Sales Dashboard (Excel)

## Overview
An Excel-based data analysis and dashboard project exploring **customer demographics and purchasing behavior** for bike sales. The project combines data cleaning, feature creation, and interactive visualizations to analyze patterns across income, age, commute distance, education, and region.

---

## Data Preparation
The raw dataset was cleaned and standardized in Excel, including:
- Converting coded values (e.g., `M`, `S` → Married / Single)
- Standardizing gender values (`M`, `F` → Male / Female)
- Correcting misspellings and inconsistent text
- Creating **age brackets** using nested IF statements
- Formatting and standardizing categorical fields

---

## Dashboard Insights
The dashboard analyzes bike purchases by:
- Average income per purchase
- Customer age brackets
- Commute distance
- Education level
- Marital status, region, and occupation (via slicers)

Interactive slicers allow users to filter results dynamically.

---

## Tools & Skills Used
- Microsoft Excel
- Data cleaning & transformation
- Nested IF formulas
- Pivot tables & pivot charts
- Dashboard design and slicers
