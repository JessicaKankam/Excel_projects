# Three-Statement Financial Model (Excel)

## Overview
An Excel-based financial analysis project building an integrated **Income Statement, Balance Sheet, and Cash Flow Statement**. The model links core financial statements and applies structured assumptions to evaluate company performance and financial health.

## What This Model Includes

- Integrated 3-statement financial model
